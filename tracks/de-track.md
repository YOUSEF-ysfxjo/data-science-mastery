# 🏗️ Track: Data Engineering

> **حقيقة:** Data Engineers يقبضون أكثر من Data Scientists في كثير من السوق. الطلب ثابت، المنافسة أقل.

---

## المنهج (٨ أسابيع)

### Week 1–2: SQL متقدم
- Window functions بعمق
- Query optimization
- Indexes, partitions
- PostgreSQL, MySQL

### Week 3: ETL/ELT
- Pandas للـ ETL الصغير
- Apache Airflow
- dbt (data build tool)
- Singer / Meltano

### Week 4: Big Data
- Apache Spark (PySpark)
- Hadoop concepts (للسياق فقط)
- Distributed computing basics

### Week 5: Cloud Data Warehouses
- BigQuery (الأنسب للبدء)
- Snowflake
- Redshift

### Week 6: Streaming
- Apache Kafka
- Kafka Streams / Flink (مقدمة)

### Week 7: Data Modeling
- Dimensional modeling (Kimball)
- Star schema, snowflake schema
- Slowly changing dimensions

### Week 8: Modern Stack
- Fivetran + dbt + Snowflake + Looker
- Lakehouse architecture (Databricks, Iceberg)
