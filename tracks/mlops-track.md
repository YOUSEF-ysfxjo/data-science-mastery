# ⚙️ Track: MLOps

> **Reality:** The market needs MLOps more than Data Scientists. If you combine both skill sets, you are **rare**.

---

## Curriculum (10 weeks)

### Week 1–2: Containers + APIs
- Docker in depth
- FastAPI for ML
- Pydantic, async
- Reverse proxies (nginx)

### Week 3: CI/CD
- GitHub Actions
- Pre-commit hooks
- Automated testing for ML code

### Week 4: Cloud Basics
- AWS (S3, EC2, Lambda, SageMaker)
- Or GCP / Azure
- IAM + cost management

### Week 5: Experiment Tracking
- MLflow
- Weights & Biases
- DVC for data versioning

### Week 6: Orchestration
- Airflow
- Prefect (modern alternative)
- Kubeflow

### Week 7: Model Serving
- BentoML
- Triton Inference Server
- TorchServe
- ONNX

### Week 8: Monitoring
- Evidently AI (data drift)
- Prometheus + Grafana
- Logging best practices

### Week 9: Feature Stores
- Feast
- Tecton

### Week 10: Comprehensive Project
End-to-end pipeline integrating everything above.
